# 智学优教 - 大学生家教O2O一站式服务平台 这是一个基于 Next.js 和 Tailwind CSS 构建的现代化 Web 应用。 ## 功能特点 - 响应式设计，支持各种设备 - 现代化的用户界面 - 用户认证系统 - 学生/教师角色切换 ## 技术栈 - Next.js 14 - React 18 - TypeScript - Tailwind CSS - Heroicons ## 开始使用 1. 安装依赖： ```bash npm install ``` 2. 运行开发服务器： ```bash npm run dev ``` 3. 打开 [http://localhost:3000](http://localhost:3000) 查看应用。 ## 项目结构 ``` zhixueyoujiao_web/ ├── app/ # Next.js 应用目录 │ ├── page.tsx # 主页面 │ ├── layout.tsx # 根布局 │ └── globals.css # 全局样式 ├── public/ # 静态资源 ├── package.json # 项目配置 └── README.md # 项目文档 ``` ## 开发指南 - 使用 `npm run dev` 启动开发服务器 - 使用 `npm run build` 构建生产版本 - 使用 `npm run start` 运行生产版本 ## 贡献指南 1.
